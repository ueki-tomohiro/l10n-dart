## Installation

> Clone this repository: `git clone https://github.com/ueki-tomohiro/l10n-dart`

### Open the directory and run the script line:

```bash
cd 10n-dart 
```
```bash
npm i  # or yarn
```
```bash
rm -rf .git && git init && git add . && git commit -m "Initial commit" #Optional
```
